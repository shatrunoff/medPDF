ТЗ консольного приложения мед анализов на go

1. есть много фото различных анализов и результатов обследований
2. есть группы по медицине: например щитовидная железа, желудок и тд. Либо по врачебной специализации можно делать.
3. фото анализов может прийти в любом фото формате (heic, jpg, mpeg … )
4. консольное приложение принимает путь до фото, специализацию и дату этого анализа на фото в формате 01-01-2001
5. фото помещается в папку foto в поддиректорию специализации (например эндокринология )
6. фото переименовывается в формат направление_01_01_2001.обобщенный формат фото
7. из этих фото генерится pdf в директорию pdf, где в возрастающей по дате сортировке все фото для определенной специализации
